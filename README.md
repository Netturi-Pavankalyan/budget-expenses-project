# 💰 FinTrack

A modern full-stack personal finance application that helps users manage budgets, track expenses, monitor multiple bank accounts, and analyze spending—all in one place.

---

## 🚀 Overview

FinTrack is designed to simplify personal finance management with a clean interface and powerful features. It enables users to create monthly budgets, record expenses, monitor account balances, and gain valuable insights through interactive analytics.

---

## ✨ Features

* 💰 Smart expense tracking
* 📊 Monthly budget management
* 📅 Calendar-based expense history
* 🏦 Multi-account support
* 📈 Interactive analytics dashboard
* 💳 Automatic account balance updates
* 🔒 Secure JWT authentication
* 📱 Fully responsive design

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/Pavankalyan-Madiga/budget-expenses-project.git
cd budget-expenses-project
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```
